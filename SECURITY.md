# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Conduit, please report it responsibly.

**Do not open a public issue.** Instead, use one of the following:

- [GitHub Security Advisory](../../security/advisories/new) (preferred)
- Email: tanishqchawla7@gmail.com

Include a description of the vulnerability, steps to reproduce, and the potential impact.

You should receive a response within 72 hours. Confirmed vulnerabilities will be patched and disclosed after a fix is available.
